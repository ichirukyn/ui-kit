{{#markdown}} ###### Немного о реализации выпадающих списков Для селекторов используется библиотека `choices js`, на данный момент она выглядит наиболее удобной в плане подгонки стилей, а так же по своим возможностям. > Но это не значит, что нельзя использовать любой другой. В отличии от других элементов, у выпадающего списка 2 варианта добавления цветов. `sub` -- Для постоянного цвета бордера и выбранных элементов мультиселектора `sub-active` -- Модификатор, срабатывает при открытии выпадающего списка Так как `choices js` инициализирует всё вокруг `select`, то для прокидывания стилей в "родителя" используется небольшой костыль в виде `data-style=''` ###### Примеры Пример работы модификатора: Постоянный цвет, удобно для ошибок: ###### Мультивыбор ###### Размеры ###### Селектор для ссылок..
