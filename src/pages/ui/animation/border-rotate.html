{{#markdown}} `block_border` Является основным классом, для инициализации необходимы стилей, модификаторы же необходимы для запуска анимаций и корректировки некоторых стилей (прим. `block_border-rotate-ccw`) По умолчанию весь контент располагается по центру, для удобства выравнивание градиента-подложки. Для выравнивания по нужному краю, рекомендуется сделать вложенный блок с заполнением ширины и высоты по родителю. Для фиксированных размеров родителя, в противном случае можно не указывать размеры и всё будет автоматически подгоняться под контент.
