{{#markdown}} ### Аккордион ##### Введение Компонент Accordion позволяет пользователям показывать и скрывать разделы связанного контента на странице. Accordion состоит из нескольких важных частей: ###### `.accordion` Является обёрткой всего аккордиона и является "селектором-якорем" для иницилаизации. По умолчанию без transition, но можно расширить с помощью `.accordion_transition`. > Не рекомендуется задавать `gap` текущему классу, чтобы не было проблем с отображением панели. ###### `.accordion__open` Заголовок аккордиона и по совместительству является "большой кнопкной". При необходимости можно сделать заголовок и кнопку отдельно, просто класс нужно повесить на необходимый элемент. Обычно предпологается что заголовок будет со стрелочкой, поэтому для неё есть специальные классы модификаторы, которые сработают при открытии (то есть, при `.accordion__open.active`): - `.arrow_rotate_180` - Поворачивает стрелочку на 180deg - `.arrow_rotate-180` - Поворачивает стрелочку на -180deg - `.arrow_mirrorX` - Отражает стрелочку по X - `.arrow_mirrorY` - Отражает стрелочку по Y ###### `.accordion__panel` Внутренний контент аккордиона. По умолчанию статичный, но можно легко повесить absolute. > Можно вкладывать аккордионы в аккордионы, главное не запутаться в 'data-acc' ###### `data-acc=id` Нужен для разграничения аккордионов на страницы и особенно для вложенных аккордионов. ##### Примеры

Простой аккордион

any content

Простой пример компонента, вынесенный в partials: {{#> accordion id='2' title='Простой аккордион 2' arrow-style='arrow_rotate_180'}}

any content

{{/accordion}}

###### Аккордион со стилями В этом примере мы добавили стили, но чтобы было удобнее разделять стили блока и компонента. Так же добавили плавное открытие за счёт `.accordion_transition`.
{{#> accordion wrapper-style='accordion_transition w_100p' id='3' title='Аккордион' arrow-style='arrow_rotate_180' outside=''}}

any content

{{/accordion}}
